Available drivers for Transportation Project and Tasks aplication

This application additionally helps the supervisor or dispatcher to find out the availability of drivers assigned to a specific project in the Odoo Transportation project application.

Overview

Task section of project

fill up the fields Journey Start Date and Journey End Date

Check the availability of drivers for a specific date and time.

Under Under the Reporting menu, go to Drivers and select the desired date and time to check availability.

Result of the search

The search found available drivers for the requested period

See details for the certain driver

Click on the driver of your choice to view their details.

This module is an additional component of the Transportation project

Bus companies, taxi services, and personal transportation companies utilize this system to plan and manage driver assignments, organize work into projects, generate work orders, enable drivers to record timesheets, and bill clients directly for their services.
